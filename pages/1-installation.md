#### Build and Install

On OS X, Linux, etc., use `autobuild.sh` or:

```sh
perl Build.pl
./Build
./Build test
./Build install
```

On Windows, use `autobuild.bat` or:

```sh
perl Build.pl
Build
Build test
Build install
```
